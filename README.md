# RPG_IDLE

GitHub Pages로 바로 띄울 수 있는 방치형 RPG 프로토타입입니다.

## 현재 포함된 루프

- `100 x 100` 구조의 월드 / 세부 스테이지 진행
- 일정 월드 도달 시 던전 해금
- `Gold`, `Diamond`, `Essence` 3종 재화
- `Diamond` 기반 장비 뽑기
- `9부위 장비` 시스템
  - 투구, 방어구, 무기, 반지, 목걸이, 팔찌, 각반, 장갑, 신발
- 장비 등급 체계
  - 일반, 고급, 희귀, 영웅, 전설, 신화, 유일
- 장비 합성 시스템
  - 같은 부위, 같은 등급 장비 3개로 상위 등급 장비 제작
  - 유일 장비 3개 합성 시 뽑기로는 나오지 않는 `창조 아이템` 제작
- 업그레이드 / 축복 설명 UI
- 장비창 / 인벤토리 / 합성 / 유물 / 던전 / 스테이터스 / 로그 메뉴
- 로컬 저장과 오프라인 보상

## 구조

- 원본 소스: `src/app.ts`
- 브라우저 실행 파일: `app.js`
- 스타일: `styles.css`
- 정적 진입점: `index.html`

현재 작업 환경에는 `node`와 `tsc`가 없어서, 브라우저에서 직접 읽는 `app.js`를 함께 유지합니다.

## 저장 방식

플레이 데이터는 브라우저의 `localStorage`에 저장됩니다.

- 같은 브라우저와 같은 기기에서 유지됩니다.
- 다른 기기와 자동 동기화되지는 않습니다.
- 계정 저장, 랭킹, 기기 간 동기화가 필요하면 이후 백엔드가 필요합니다.
